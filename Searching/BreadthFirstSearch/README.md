# Breadth First Search

-----------

```bash
                .-~~~~~~~~~-._       _.-~~~~~~~~~-.
            __.'   Breadth    ~.   .~              `.__
          .'//     First        \./                  \\`.
        .'//       Search        |                     \\`.
      .'// .-~"""""""~~~~-._     |     _,-~~~~"""""""~-. \\`.  
    .'//.-"                 `-.  |  .-'                 "-.\\`.
  .'//______.============-..   \ | /   ..-============.______\\`.
.'______________________________\|/______________________________`.  
      
      MAX-HEAPIFY(A,i,heap-size)

      1 l = LEFT(i)
      2 r = RIGHT(i)
      3 if l <= heap-size and A[l] > A[i]
      4   largest = l
      5 else largest = i
```
